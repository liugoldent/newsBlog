# CSS 基本知識
## 區塊元素 vs 行內元素
### 常見
常見區塊元素標籤：div、ul、li、p、h1
常見行內元素標籤：span、a、input、img
### 差別
#### inline
* 元素可在同一行內呈現、圖文均不換行，也不會影響版面配置
* 不可設定長寬，寬高由內容撐開
#### block
* 元素寬度預設撐到最大，佔滿整個容器
* 可設定長寬
## 滿版配置
* h-screen > main part（設定 h-max）> footer 就會自動置底
